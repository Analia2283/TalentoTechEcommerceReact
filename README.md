# Proyecto Ecommerce con React

## Descripción del proyecto. Venta de productos, comprende las siguientes tecnologías.

- React


### Funcionalidades Principales

- El proyecto incluye las siguientes características:

- Catálogo de Productos: Visualización de todos los productos y filtros por categoría.

- Detalle del Producto: Vista individual de cada artículo con descripción, precio y opción de agregar al carrito.

- Carrito de Compras: Gestión de productos seleccionados, cantidades y total.

- Checkout: Proceso de pago simulado y generación de órdenes de compra.

- Autenticación de Usuarios: Login/Logout de clientes y administradores.

- Ruta Protegida (Admin): Dashboard exclusivo para el usuario administrador.

- Búsqueda Dinámica: Componente de barra de búsqueda para filtrar productos en tiempo real. 


#### Instalación
1. Versiones d software.
    - Node >= 18.0.0
    - Vite >= 3.1.4

2. Instalar dependencias

```
npm install
```



4. Levantar el proyecto

```
npm run dev
```

### Librerías implementadas
